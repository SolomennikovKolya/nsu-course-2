_mm_mul_ps:
Принимает: Два SSE регистра, каждый содержит четыре значения типа float.
Возвращает: SSE регистр, результат умножения соответствующих элементов входных регистров.

_mm_add_ps:
Принимает: Два SSE регистра, каждый содержит четыре значения типа float.
Возвращает: SSE регистр, результат сложения соответствующих элементов входных регистров.

_mm_storeu_ps:
Принимает: Указатель на память и SSE регистр с четырьмя значениями типа float.
Возвращает: Ничего. Записывает значения из SSE регистра в память по указанному адресу.

_mm_div_ps:
Принимает: Два SSE регистра, каждый содержит четыре значения типа float.
Возвращает: SSE регистр, результат деления соответствующих элементов первого регистра на соответствующие элементы второго регистра.

mm_set1_ps:
Принимает: Одно значение типа float.
Возвращает: SSE регистр, в котором все элементы равны переданному значению.

_mm_sub_ps:
Принимает: Два SSE регистра, каждый содержит четыре значения типа float.
Возвращает: SSE регистр, результат вычитания соответствующих элементов второго регистра из элементов первого регистра.

cblas_scopy:
Принимает: Размер n (количество элементов), указатель на массив x, шаг incx, указатель на массив y, и шаг incy.
Возвращает: Ничего. Копирует элементы из массива x в массив y.

cblas_saxpy:
Принимает: Размер n (количество элементов), значение alpha, указатель на массив x, шаг incx, указатель на массив y, и шаг incy.
Возвращает: Ничего. Выполняет операцию y = alpha * x + y.

cblas_sgemm:
Принимает: Строковый порядок Order, транспонирование матриц TransA и TransB, размеры матриц M, N и K, значение alpha, указатель на матрицу A, локальный размер lda, указатель на матрицу B, локальный размер ldb, значение beta, указатель на матрицу C, и локальный размер ldc.
Возвращает: Ничего. Выполняет умножение матриц A и B с использованием заданных параметров и сохраняет результат в матрице C.

_mm_loadu_ps
Принимает: Указатель на память, откуда будут загружены значения типа float.
Возвращает: SSE регистр, содержащий значения, загруженные из указанной памяти.
